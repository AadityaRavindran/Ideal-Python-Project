and https://packaging.python.org/en/latest/guides/using-manifest-in/ 

## Usage
To run the scripts from the module,
1. cd \<the folder containing the project>
2. pip install -e .
3. ring-lord

## Testing
To run the module tests,
1. cd \<the folder containing the project>
2. python3 setup.py test
